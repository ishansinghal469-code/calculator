# Calculator

A clean and fully functional calculator web application built with 
HTML, CSS, and JavaScript that performs all basic arithmetic operations.

---



##  Features

- **Basic Operations** — Addition, Subtraction, Multiplication, Division
- **Chained Calculations** — Use result of one calculation in the next
- **Divide by Zero Protection** — Shows error instead of `Infinity`
- **Decimal Support** — With double decimal prevention
- **Full Keyboard Support** — Use your keyboard to calculate
- **Backspace Support** — Delete last digit easily
- **AC Button** — Clear everything and start fresh
- Fully **Responsive** for mobile and desktop

---

##  Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Calculator Structure & Layout |
| CSS3 | Styling, Grid & Animations |
| JavaScript | Calculator Logic & Keyboard Support |
